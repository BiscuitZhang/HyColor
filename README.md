# HyColor: An Efficient Heuristic Algorithm for Graph Coloring

This repository contains the executable files for the paper "HyColor: An Efficient Heuristic Algorithm for Graph Coloring". The source code will be updated in the future.

## Overview

HyColor is an efficient heuristic algorithm designed for the graph coloring problem.

## Quick Start

### Prepare Datasets

The expected directory structure is as follows:

```
.
├── datasets/
│   ├── DIMACS/
│   │   ├── graph1.graph
│   │   ├── graph2.graph
│   │   └── ...
│   ├── SNAP/
│   │   ├── email-Eu-core.graph
│   │   └── ...
│   └── ...
├── color                 (The executable file)
└── run_hycolor.sh        (The run script)
```

### Add Execute Permissions

Before running, you need to make the files executable:

```
chmod +x color
chmod +x run_hycolor.sh
```

### Run the Algorithm

```
./run_hycolor.sh
```
